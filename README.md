## アプリ名. 
TreNavi
## アプリの概要. 
理想のパーソナルトレーナーを探して予約するアプリ(モバイル想定). 
## アプリの詳細. 
TreNavi診断により理想のトレーナータイプを診断し、自分に合うトレーナーとマッチできる. 
## アプリの機能. 
### クライアントサイド. 
- ログイン機能. 
- 検索機能. 
- いいね機能. 
- 予約機能. 
- レビュー機能. 
- 理想のトレーナータイプ診断（TreNavi診断）. 
- マッチ度を5点満点表示（小数点第一位まで）. 
### トレーナーサイド. 
- トレーナー登録. 
- プラン登録. 
- 予約ステータス変更. 
## プロダクトに対する想い. 
- パーソナルトレーニングでは、初対面のトレーナーに対して不安を感じたり、そもそも選定の基準が分からないという課題があると感じました. 
- 一般的なECサイトは総合的な5段階評価が中心であり、「評価基準の曖昧さ」「評価は高いが自分には合わない」といったミスマッチが起こりやすいと考えています. 
- そこで本プロダクトでは、トレーナーの指導スタイルや人柄を複数の観点で可視化し、ユーザー自身の価値観と照らし合わせて選択できる仕組みを提供することを目指しました. 
- これにより、トレーナー選びにおける不安を軽減し、より納得感のあるマッチングの実現を目指しています. 
## 工夫した点、こだわった点. 
- パーソナルトレーナーに対する口コミを「指導の厳しさ」「科学的or感覚的」など5つの観点で5段階評価として収集し、各項目の平均値からトレーナーの特徴を数値化・可視化しました. 
- ユーザーも同様の質問に回答することで、自身の好みや理想のトレーナー像を数値として把握できる診断機能を実装しました. 
- レーナーとユーザー双方の評価データをもとに、各項目の差分からマッチ度を算出し、相性の良いトレーナーを提案できるようにしました.
